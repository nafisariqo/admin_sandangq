# Sandangq

Sandangq adalah sebuah website donasi adalah wadah/platform untuk berdonasi sandang secara online. Website ini menghubungkan admin dengan para pendonasi yaitu melalui fitur donasi dan publikasi berita. Fitur
donasi memungkinkan pendonasi untuk melakukan donasi sandang serta memantau donasi yang masuk ke admin. Begitu juga dengan pihak admin, admin dapat memantau, memvalidasi donasi yang masuk dan. Sementara itu,
disertakan juga fitur publikasi berita yang dirancang bagi admin untuk melakukan publikasi pengumuman maupun kegiatan yang akan dilakukan sehingga para pendonasi maupun pengunjung situs dapat mengetahui bagaimana pemanfaatan donasi yang masuk ke dalam website.

### Technology i have used in this app :
  1.  Laravel 7
  2.  Bootstrap 
  3.  Jquery
  4.  Php MyAdmin
  5.  Sweetalert
  6.  Summernotes
